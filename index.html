From Wikipedia:
Password strength is a measure of the effectiveness of a password against guessing or brute-force attacks. In its usual form, it estimates how many trials an attacker who does not have direct access to the password would need, on average, to guess it correctly. The strength of a password is a function of length, complexity, and unpredictability.
This website implements that function spoken of.
This website will never store your password information, as all calculations are done on the client's end with no communication to a backend. This site is hosted using GitHub Pages, and thus there is not even a server to communicate with. If you wish to inspect the code yourself, it is hosted for free under an MIT license at kyle-west/password-evaluator . Feel free to check it out.
